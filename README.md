# Манифест Градусной Алгебры (Degree Math)

Реализация альтернативной математической системы, расширяющей пространство вещественных и комплексных чисел через механизм градусных маркеров (*) и мнимых градусов (i). 

Система полностью сохраняет фундаментальные законы классической алгебры, включая 100% выполнение дистрибутивности: 
a • (b + c) = a • b + a • c
Это делает её применимой в линейной алгебре, математическом анализе и задачах отказоустойчивого программирования.

## Ключевые Аксиомы и Законы

### 1. Закон Единичного Градуса и Инкапсуляция Знака
Классические отрицательные числа переопределяются как числа с градусным маркером:
-a = a*

В отличие от классического минуса, градусный маркер является внутренним флагом состояния числа. При возведении в степень выполняется закон:
(a*)^b = a^(b • 1*)

### 2. Закон Радикала и Мнимый Градус (i)
Извлечение корня из градусного (отрицательного) пространства порождает мнимый градус i:
√a* = √a • i

При этом система поддерживает внутреннюю цикличественность маркеров:
i • i = 1*

### 3. Закон Ориентированного Нуля (Сингулярная Устойчивость)
Деление на нуль не вызывает исключений в коде, а возвращает ориентированную бесконечность, сохраняющую свойства пространства:
a / 0* = ∞*

Это позволяет строить сквозные непрерывные вычисления (Zero-Crash Computation) в физических и геометрических симуляторах.

### 4. Закон Шахматного Градуса в Матрицах
В многомерных пространствах классическое чередование знаков (плюс/минус) заменяется градусным разворотом (умножением на 1*). При расчете детерминанта 3х3 применяется Шахматный Флип, сохраняющий однородность операций.

## Как это работает в коде (Quick Start)

Библиотека написана на чистом Python и не имеет внешних зависимостей. Благодаря инкапсуляции знаков в битовые флаги, вычисления выполняются через быстрые логические операции XOR.

```python
from degree_math import DegreeNumber, DegreeSolver

# Автоматический обход классических ограничений логарифма
# Решаем уравнение: 2^x = 8* (в обычной математике решения нет)
base = DegreeNumber(2)
result = DegreeNumber(8, is_degree=True)

x = DegreeSolver.solve_exponential(base, result)
print(f"Решение: x = {x}")  # Выведет: 3.0*
print(f"Проверка: 2 ** 3.0* = {base ** x}")  # Выведет: 8.0*
```
